This repository (`e-plantShopping`) contains a React shopping cart application for an online plant shop called **Paradise Nursery**.

### Features
- Landing page with a **Get Started** button
- Navigation bar with links to **Landing**, **Plants**, and **Cart**
- Product listing page with plant cards (image, name, description, price, add-to-cart)
- Plant categories/sections (e.g., Aromatic Plants, Medicinal Plants)
- Cart page with:
  - Item cards showing thumbnail, unit price, quantity controls, per-item total
  - Delete item option
  - Updated cart count and total price
  - Continue Shopping and Checkout buttons

### Tech Stack
- React
- Redux Toolkit (state management)
- CSS
